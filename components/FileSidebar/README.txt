# FileSidebar Modular Components

These are reusable, production-ready components extracted from FileSidebar.tsx with all security fixes applied.

## Components
- **types.ts** - TypeScript definitions
- **utils.ts** - Helper functions (removeFolderHelper, buildFileTree, getIcon, sanitizeHtml)
- **ConfirmModal.tsx** - Modal for confirmations
- **FileContextMenu.tsx** - Context menu component
- **PdfPageRenderer.tsx** - PDF page rendering
- **FilePreviewViewer.tsx** - File preview with security fixes

## Usage
These components can be imported individually for reuse in other parts of the application.

## Current Status
The main FileSidebar.tsx (parent folder) is the active, working file with all fixes applied.
These modular components are available for future refactoring or reuse.
