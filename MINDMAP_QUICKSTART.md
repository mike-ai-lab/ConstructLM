## âš ï¸ Troubleshooting

### "Failed to generate mind map"
- **Check**: API key is set in Settings
- **Try**: Different AI model
- **Solution**: Smaller file or simpler content

### Mind map looks empty
- **Reason**: File has no clear structure
- **Try**: Different file type
- **Solution**: Use structured documents

### Loading takes too long
- **Reason**: Large file or slow model
- **Try**: Faster model (Gemini Flash)
- **Solution**: Wait or use smaller file

### Can't see all nodes
- **Solution**: Zoom out (scroll down)
- **Solution**: Click parent nodes to collapse
