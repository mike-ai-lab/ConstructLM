# Standalone Dictation App

This folder contains the isolated dictation feature for future development.

## Files
- `DictationTab.tsx` - Main dictation component
- `useLiveDictation.ts` - Live dictation hook

## Dependencies
- Requires: `geminiService.ts`, `audioUtils.ts`, `chatRegistry.ts`
- Icons: lucide-react

## Future Development
This can be developed as a standalone application independent of the main ConstructLM app.
